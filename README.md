> "I build AI tools I actually ship. The work speaks now."

## What This Is

No Mouth Labs is Rob Spence's AI project portfolio — a curated showcase of real tools built out of necessity, not for demo purposes. Every project on the site is fully shipped and actively used. The portfolio spans agentic systems, developer tools, LLM-powered applications, and product experiments.

## Who Built It

**Rob Spence** — Solutions Architect by day, AI tool builder by compulsion. He builds with Claude, ships with Vercel, and documents nothing (until now). The projects here are his proof of work.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI**: shadcn/ui + custom components
- **Runtime**: React 19
- **Deployment**: Vercel
